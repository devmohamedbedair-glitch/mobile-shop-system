# 📱 Mobile Shop System

A simple ASP.NET Core MVC application for managing a mobile accessories shop.

## 🚀 Features
- Manage Products (CRUD)
- Create Invoices (Multiple Products)
- Auto calculate total price
- Reduce stock after sale
- Invoice details & printing

## 🛠️ Technologies
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
